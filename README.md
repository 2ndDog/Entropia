# Entropia
![image](https://raw.githubusercontent.com/2ndDog/Entropia/master/Entropia.svg)


Entropia 来自于 Entropy 的意大利语，意为“熵”，我们的计算机视觉技术意味着提高图像的熵。


#
基于RDDLN(残差密集深度分级网络, Residual Dense Deep-Levels Network)的单图像超分辨率

⚠GPU版本需CUDA-11.1
